## Overview

The **Document Service** handles generation and storage of official insurance documents in PDF format. It provides secure storage with access control and digital signature support.

**Actors:** Clients, Agents, Brokers, Back-office staff.

**Key features:**
- PDF generation (policy contracts, endorsements, cancellation letters)
- Secure storage with access control
- Digital signatures and timestamping
- Retrieval API for clients and brokers
