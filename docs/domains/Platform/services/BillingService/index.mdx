## Overview

The **Billing Service** is responsible for generating invoices and managing premium installment plans for active Policies.

**Actors:** Clients, Back-office financial team.

**Key features:**
- Invoice generation based on policy premiums
- Installment plan management (monthly, yearly, custom)
- Invoice lifecycle (issued → paid / overdue / canceled)
- Automatic overdue detection
- Integration with external ERP if needed
