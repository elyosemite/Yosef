## Overview

The **Notification Service** acts as an event consumer and dispatcher for all outbound communications. Built with TypeScript/Node.js to handle async I/O and third-party API integrations efficiently.

**Actors:** Clients, Brokers, Agents.

**Key features:**
- Email / SMS / Push / WhatsApp integration
- Message templating
- Retry and queueing for failed notifications
- Per-user multi-channel preferences
