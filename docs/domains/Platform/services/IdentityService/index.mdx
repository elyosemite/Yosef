## Overview

The **Identity Service** is the entry point for all user identity and permissions. It integrates with **KeyCloak** (OAuth2/OIDC) for authentication and enforces role-based access control across the platform.

**Actors:** Brokers (companies), Agents (sellers), Insureds (policyholders), System administrators.

**Key features:**
- User registration (Brokers, Agents, Insureds)
- OAuth2/JWT authentication via KeyCloak
- Role-based access control (RBAC)
- Multi-tenant support for multiple Brokerages
- Audit logging for compliance
