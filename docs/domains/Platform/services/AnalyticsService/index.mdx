## Overview

The **Analytics Service** consumes domain events from across the platform and aggregates them into a data warehouse for reporting and dashboards.

**Actors:** Executives, Brokers, Analysts.

**Key features:**
- Event aggregation and storage (data warehouse)
- Dashboard APIs (KPIs, conversion rates, loss ratios)
- Predictive analytics integration (ML pipelines)
- Broker-level analytics for performance monitoring
