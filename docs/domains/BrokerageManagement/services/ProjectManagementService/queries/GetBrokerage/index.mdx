## Overview

Retrieves a single **Brokerage** by its UUID identifier. Returns the brokerage's legal name, CNPJ, contact email, and optional phone number.

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `brokerageIdentifier` | `uuid` | ✓ | The unique identifier (GUID) of the brokerage |

## Response

**200 OK** — brokerage found.

```json title="Response body example"
{
  "identifier": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "brokerageName": "Corretora Exemplo Ltda",
  "cnpj": "12345678000195",
  "email": "contato@corretora.com.br",
  "phone": "11999990000"
}
```

**404 Not Found** — no brokerage with the given identifier exists.
