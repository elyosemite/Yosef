## Overview

Emitted by the **Project Management Service** when a Brokerage is successfully created. Previously named `OrganizationCreatedEvent` in the codebase — rename on sight.

Downstream consumers (e.g., EventProcessor, Identity Service) should listen to this event to perform initial setup for the new Brokerage (e.g., creating a KeyCloak realm or tenant).
