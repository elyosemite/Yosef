## Overview

Emitted when a Project (insurance product line) is created inside a Brokerage. Projects scope Quotations and Policies — a Quotation must reference a Project.

Examples of Projects: "Bike Insurance", "Ship Insurance", "Aircraft Insurance".

## Payload

```json title="Example payload"
{
  "projectId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "brokerageId": "3fa85f64-5717-4562-b3fc-2c963f66afa6",
  "name": "Bike Insurance",
  "description": "Coverage for bicycles and e-bikes",
  "createdAt": "2025-01-20T11:00:00Z"
}
```
