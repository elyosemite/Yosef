## Overview

The **Policy Service** is the source of truth for all insurance contracts. A Policy is created from an approved Quotation and progresses through a defined state machine.

**Actors:** Clients, Agents, Brokers, Back-office staff.

**Policy state machine:**
```
Pending → Active → Suspended → Canceled
                ↘ Renewed
                ↘ Endorsed
```

**Key features:**
- Policy issuance from accepted quotations
- Endorsement workflows (changes in coverage or insured objects)
- Renewal processing
- Integration with Billing and Document services
